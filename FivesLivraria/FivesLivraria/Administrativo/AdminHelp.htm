Visão Geral

    O módulo administrativo é de uso exclusivo dos usuários com permissão "gestor", para o controle e gerenciamento dos recursos utilizados pela livraria. Para acessar este módulo, é necessário efetuar a autenticação no módulo de login do usuário. A aplicação, ao identificar um usuário do tipo "gestor", vai direcioná-lo automaticamente para o módulo administrativo. A seguir, será detalhado cada um dos módulos, para um maior detalhamento das suas funcionalidades.
